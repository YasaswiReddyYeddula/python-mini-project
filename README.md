# Project Title : Mini Project With Iris Dataset

## Objective :
The objective of this project is to perform basic data analysis and visualization using the Iris dataset. It demonstrates how to use Python libraries to load, analyze, manipulate, and visualize data through different types of charts.

## Tools Used :
--Python
--NumPy
--Pandas
--Matplotlib
--Plotly
--VS Code 


### Project Tasks
-Loaded the Iris dataset.
-Performed basic data exploration.
-Converted DataFrame columns into NumPy arrays.
-Analyzed flower species distribution.
-Created visualizations using Matplotlib:
     -Line Chart
     -Bar Chart
     -Pie Chart
     -Histogram
     -Scatter Plot
-Created interactive visualizations using Plotly.
-Saved generated charts as image files.


## Key Findings :
--The Iris dataset contains 150 flower samples.
--There are 3 flower species:
--Setosa
--Versicolor
--Virginica
--Each species contains 50 samples.
--Data visualization helps understand the distribution of sepal and petal measurements.
--Plotly provides interactive charts, while Matplotlib is useful for creating static visualizations.
